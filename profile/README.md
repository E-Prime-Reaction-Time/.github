# E‑Prime Software — Psychology Experiment Design Environment

## About E‑Prime Software

E‑Prime Software provides a complete experiment design environment for psychology and neuroscience research with millisecond precision, reaction time recording, and multi‑modal stimulus presentation. E‑Prime Software combines text/image/audio/video stimuli, EEG triggers, and response pad support. E‑Prime Software supports reaction time measurement, accuracy scoring, response termination, and data export to Excel/SPSS directly into the interface. E‑Prime Software integrates fMRI synchronization, eye‑tracking integration, scriptable procedures, and stimulus‑locked event logging. E‑Prime Software enables users to design behavioral experiments, collect response times, and analyze performance data. E‑Prime Software runs on portable workspaces and dedicated laboratory stations. E‑Prime Software is trusted by cognitive psychologists, neuroscientists, behavioral researchers, and human factors specialists worldwide.

## Deep Dive — Experiment Design Notes

Millisecond timing accuracy vs frame‑based display, refresh rate synchronization, and when to use event logging
Stimulus presentation: text (font size/color/position), images (BMP/JPG/PNG/GIF), sounds (WAV/MP3), video (AVI/MP4), slide transitions, fixation crosses, and inter‑stimulus intervals (ISI/jittered)
Response collection: keyboard (specific keys, any key), mouse (button press/move/click), response box/pad (serial/USB/button latency), reaction time (ms precision), accuracy (correct/incorrect), response termination (timeout after X ms), and response‑locked stimulus triggers
EEG/fMRI integration: send TTL pulses at stimulus onset, response onset, block start/end; record stimulus codes to BrainVision, Neuroscan, NetStation, Biopac, and EGI systems via parallel port/USB‑to‑TTL
Scripting: E‑Basic (VBA‑derived) for procedural logic: random assignment, counterbalancing, adaptive staircases, branching, nested trials, and online performance feedback
